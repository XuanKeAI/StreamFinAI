StreamFinAI是基于InternLM2-7B模型构建的金融财务分析智能体，能基于金融新闻、研报和相关政策文件为用户提供投资建议、时事分析和政策解读，并能快捷迅速的分析和处理财务会计数据，提取和概括有效信息，辅助决策。
